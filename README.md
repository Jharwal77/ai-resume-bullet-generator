# 🚀 AI Resume Bullet Generator

An AI-powered web application that generates **ATS-friendly resume bullet points** from project details using **Google Gemini AI**.

Built with **React, Node.js, Express, and Gemini API**, this tool helps students and developers quickly create professional resume descriptions for projects and internships.

---

## ✨ Features

* 🤖 AI-generated resume bullet points
* 📄 Generates exactly 5 ATS-friendly bullet points
* 💼 Optimized for Software Engineering resumes
* ⚡ Fast and responsive interface
* 📋 Copy generated bullets with one click
* 🎯 Uses strong action verbs and industry keywords
* 📱 Fully responsive design

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/ai-resume-bullet-generator.git
cd ai-resume-bullet-generator
```

---

## Backend Setup

```bash
cd server
npm install
```

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Run backend:

```bash
npm run dev
```

Server starts at:

```
http://localhost:5000
```

---

## Frontend Setup

Open another terminal:

```bash
cd client
npm install
npm run dev
```

Frontend starts at:

```
http://localhost:5173
```

---

# 📌 Usage

1. Enter Job Role
2. Enter Project Name
3. Enter Technologies Used
4. Enter Project Description
5. Click **Generate**
6. AI generates 5 ATS-friendly resume bullet points
7. Copy and use them in your resume

# 🌍 Deployment
